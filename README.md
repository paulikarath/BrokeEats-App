BrokeEats is a mobile application built with React Native and Expo designed to help students and budget-conscious users manage meal planning effectively. The app calculates meal suggestions based on a specific budget and timeframe, focusing on affordable staples and pantry-ready ingredients.

Key Features

-Intelligent Budget Planning: Dynamically calculates recipe suggestions based on user-defined financial limits and duration.

-Pantry Integration: Allows users to find recipes based on ingredients they already own to minimize food waste.

-Optimized UI/UX: Features a modern, high-contrast interface with a focus on ease of use and visual hierarchy.

-Cross-Platform Performance: Developed using TypeScript for high reliability across both iOS and Android devices.

Tech Stack

Framework: React Native (Expo)

Navigation: Expo Router

Language: TypeScript

Styling: StyleSheet API with custom modern design patterns
